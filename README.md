# Sistema de Gestão

Sistema de Gestão de Empresas, criado para praticar os conhecimentos adquiridos em desenvolvimento desktop, web e mobile. 

Com este repositório, eu pretendo inserir o máximo de conhecimentos em desenvolvimento que eu adquirir nos projetos aqui contidos.

Por enquanto ele possui uma aplicação desenvolvida em PHP vanilla com HTML, CSS, JavaScript e jQuery no front end e banco de dados MySQL e, atualmente, estou construindo uma API, também em PHP puro, para consumir e disponibilizar as informações do banco de dados utilizando o padrão REST.
