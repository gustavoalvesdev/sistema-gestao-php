# sistema-gestao-php
Sistema de gestão em PHP e MySQL, utliizando estrutura MVC Própria

## Clonar o repositório
```bash
git clone git@github.com:gustavoalvesdev/sistema-gestao-php.git
```

## Instalar as dependências
```bash
composer install
```

## Criar o arquivo .env a partir do .env.example (Windows)
```bash
copy .env.example .env
```
## Criar o arquivo .env a partir do .env.example (Linux)
```bash
cp .env.example .env
```

### Após isso, configurar o .env com as configurações do ambiente
